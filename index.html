PROJECTS
SAW, India SAW India stands for Stray Animal Welfare India. As its name suggests, this project is focused on stray animal welfare in India. It provides end users with information on all animal welfare organizations in the area if they see an animal in need. It consists of three parts, i.e. an end user application, a helper application and an admin console.
Some major features of this project include :
  • YouTube API to show related videos to users to promote animal welfare.
  • Google Firebase Realtime Database to keep track of all users and complaints.
  • Google Places And Maps API's to get details of nearby animal welfare organizations.
  • Firebase Storage to store images of injured animals.
  • Razorpay API for facilitating people to donate.
  • Firebase Authentication in order to ensure the credibility of users.
